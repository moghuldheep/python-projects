# Day 3 – Loops

## Objective

Understand how to automate repetitive tasks using Python loops.

## Topics Covered

* `for` Loop
* `while` Loop
* `range()`
* Loop Control
* Running Totals

## Tasks Completed

* Print numbers from 1 to 10
* Display even numbers
* Multiplication table generator
* Sum of first N numbers
* Countdown using `while`
* Password validation loop
* Star pattern generator

## Skills Learned

* Repeating tasks
* Iterating over ranges
* Accumulating values
* Building simple repetitive workflows

## Data Engineering Relevance

Loops are heavily used in ETL pipelines to process records one at a time, iterate through files, validate datasets, and transform millions of rows efficiently.
