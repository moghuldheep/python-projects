# Day 6 – String Methods

## Objective

Learn how to clean, manipulate, and validate text data using Python string methods.

## Topics Covered

* `upper()`
* `lower()`
* `title()`
* `strip()`
* `replace()`
* `split()`
* `join()`
* `startswith()`
* `endswith()`
* `find()`

## Tasks Completed

* Format names
* Remove extra spaces
* Validate email addresses
* Replace words in a sentence
* Split customer records
* Join city names
* Filter CSV files using file extensions

## Skills Learned

* Cleaning text data
* Transforming strings
* Splitting and joining records
* Performing basic data validation

## Data Engineering Relevance

Most real-world data arrives as text. String manipulation is essential for cleaning customer records, processing CSV files, validating inputs, and preparing data for ETL pipelines.
