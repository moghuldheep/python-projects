# Day 2 – Operators & Conditional Statements

## Objective

Learn how to perform calculations, compare values, and make decisions in Python using conditional statements.

## Topics Covered

* Arithmetic Operators
* Comparison Operators
* Logical Operators
* `if`
* `elif`
* `else`

## Tasks Completed

* Basic calculator
* Voting eligibility checker
* Even or odd number checker
* Largest number finder
* Bonus eligibility checker
* Student grading system
* Loan eligibility checker

## Skills Learned

* Decision making
* Writing conditional logic
* Comparing values
* Building simple business rules

## Data Engineering Relevance

Data Engineers frequently apply conditional logic while cleaning, validating, and transforming datasets. Conditions help identify invalid records, categorize data, and implement business rules during ETL processes.
