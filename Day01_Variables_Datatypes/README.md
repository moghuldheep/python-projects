# Day 1 – Variables, Data Types & User Input

## Objective

Learn the fundamentals of Python programming by understanding variables, different data types, user input, and formatted output.

## Topics Covered

* Variables
* Data Types (`str`, `int`, `float`, `bool`)
* User Input using `input()`
* Type Conversion
* Formatted Strings (f-strings)
* Basic Arithmetic Operations

## Tasks Completed

* Store and display personal information
* Identify Python data types using `type()`
* Accept user input and display formatted output
* Build a simple calculator
* Create a profile card using variables
* Calculate annual salary from monthly salary

## Skills Learned

* Declaring variables
* Accepting user input
* Converting data types
* Displaying formatted output
* Performing basic calculations

## Data Engineering Relevance

Variables and data types are the building blocks of every ETL pipeline. Understanding how Python stores and processes different types of data is essential before working with files, APIs, databases, and large datasets.
